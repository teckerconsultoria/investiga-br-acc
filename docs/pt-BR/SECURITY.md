# POLÍTICA DE SEGURANÇA — World Transparency Graph (WTG)

Policy-Version: v1.0.0  
Effective-Date: 2026-02-28  
Owner: WTG Governance Team

## Reporte de vulnerabilidades

Use GitHub Security Advisories para disclosure responsável:

- Caminho privado: aba `Security` do repositório -> `Report a vulnerability`.
- Não divulgue detalhes de exploração publicamente antes da triagem.

## Versões suportadas

Suporte de segurança aplica-se a:

- linha mais recente do `main`;
- release pública mais recente tagueada.

Snapshots antigos podem não receber correções.

## SLA-alvo de disclosure

Janelas alvo:

- confirmação de recebimento: até 72 horas;
- triagem inicial: até 7 dias corridos;
- plano de mitigação: assim que impacto e reprodução forem confirmados.

Esses prazos são metas de melhor esforço e não obrigação contratual.

Relatos fora de escopo ou sem reprodutibilidade terão justificativa registrada no fluxo de advisory.
